How well is the data cleaned/processed and explored? Are all steps reasonable and well explained? Are alternatives discussed and considered? Are meaningful exploratory results shown (e.g. in the supplementary materials)?

3.4.1 Feedback and Comments

The whole cleaning process is clear and well organized. The figures generated look nice and the explanation is in detailed, so I can understand them easily. 3.4.2 Summary assessment

  • some weaknesses in wrangling and exploratory component

3.5 Appropriateness of Analysis

Were the analysis methods appropriate for the data? Was the analysis done properly? Were different components of the analysis (e.g. performance measure, variable selection, data pre-processing, model evaluation) done in the best way possible and explained well?

3.5.1 Feedback and Comments

I think the linear regression models chosen are appropriate and there is no problem during data analysis. But the author might want to delete some words “3.5 Statistical Analysis:” in your manuscript which wasn’t used by the author.

3.5.2 Summary assessment

  • strong and reasonable analysis

3.6 Presentation

How well are results presented? Are tables and figures easy to read and understand? Are the main figures/tables publication level quality?

3.6.1 Feedback and Comments

The results are explained in detail and can be clearly understood. Tables and figures are presented as assisting tools. But I think it will be better if random forest model could be introduced a little bit more deeper and the figures of rf model could be improved since they only show the error changes by the number of tree, it could be better if prediction related information shown at the same time.

3.6.2 Summary assessment

  • results are presented ok, with room for improvement

3.7 Discussion/Conclusions

Are the study findings properly discussed? Are strengths and limitations acknowledged? Are findings interpreted properly?

3.7.1 Feedback and Comments

The discussion part include the finding obtained from data analysis with possible reasons of the results are also listed and analysed. I just have one small question. Since random forest model is not suitable for this data, then does the author need to suggest another better model or compared the result of random forest model with simple linear model? Strengths and limitations are acknowledged.

3.7.2 Summary assessment

  • strong, complete and clear discussion

3.8 Further comments

I am not so sure besides random forest model, there won’t be any more machine learning models could be fit to the data. So I may suggest trying other machine learning models or just using simple linear regression for prediction and compare the performance between 2 model.
